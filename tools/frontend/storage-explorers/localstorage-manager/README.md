# LocalStorage & SessionStorage Explorer X

An advanced tool for managing web storage (LocalStorage and SessionStorage).

## Features

- **Dual Storage Support**: Switch seamlessly between `localStorage` and `sessionStorage`.
- **CRUD Operations**: Add, Read, Edit, and Delete key-value pairs with ease.
- **JSON Formatting**: Built-in JSON formatter for editing complex objects stored as strings.
- **Storage Stats**: View total item count and approximate size usage.
- **Real-time Updates**: Automatically refreshes when storage changes in other tabs (for `localStorage`).

## Usage

1. Open `index.html` in your browser.
2. Select the tab for the storage type you want to manage.
3. **Add Item**: Click "Add Item", enter a key and value.
4. **Edit Item**: Click "Edit" on any row. If the value is JSON, use "Format JSON" to pretty-print it.
5. **Delete**: Remove individual items or use "Clear All" to wipe the storage.
