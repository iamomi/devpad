# Base64 Encoder/Decoder

A versatile tool for handling Base64 encoding and decoding for both text and files.

## Features

- **Text Mode**: Encode and decode plain text strings.
- **File Mode**: Upload any file (images, PDFs, etc.) to convert it to a Base64 Data URL.
- **File Download**: Convert a Base64 string back into a downloadable file.
- **Drag & Drop**: Easy file upload interface.
- **Clipboard Integration**: One-click copy for outputs.

## Usage

1. Open `index.html` in your browser.
2. **Text Mode**:
    - Type in the left box.
    - Click the "Down Arrow" (Encode) or "Up Arrow" (Decode) button in the center.
    - View result in the right box.
3. **File Mode**:
    - Switch to the "File Mode" tab.
    - Drag and drop a file to generate its Base64 string.
    - Or, paste a Base64 string in the bottom section and click "Download File" to save it.
