# JSON Formatter & Validator

A robust tool to format, validate, and minify JSON data.

## Features

- **Format**: Pretty-print JSON with customizable indentation (2 spaces, 4 spaces, or tabs).
- **Minify**: Compress JSON into a single line for production use.
- **Validate**: Real-time validation with error reporting.
- **Stats**: See the size of your JSON data in bytes/KB.
- **Line Numbers**: Easy reference for large JSON files.

## Usage

1. Open `index.html` in your browser.
2. Paste your JSON string into the editor.
3. The status bar at the top will indicate if the JSON is valid or invalid.
4. Click "Format" to pretty-print the JSON.
5. Click "Minify" to compress it.
6. Use "Copy" to copy the result to your clipboard.
