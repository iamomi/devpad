# Markdown Previewer with Export

A real-time Markdown editor with live preview and syntax highlighting.

## Features

- **Live Preview**: See your Markdown rendered instantly as you type.
- **Syntax Highlighting**: Automatic code highlighting for code blocks using `highlight.js`.
- **Toolbar**: Quick access to common Markdown formatting (Bold, Italic, Lists, Code, etc.).
- **Copy HTML**: One-click button to copy the generated HTML to your clipboard.
- **Print/PDF**: Optimized print styles to save your document as a PDF.
- **Sync Scroll**: The preview pane automatically scrolls with the editor.

## Usage

1. Open `index.html` in your browser.
2. Type Markdown in the left pane.
3. View the rendered result in the right pane.
4. Use the toolbar buttons to format text quickly.
5. Click "Copy HTML" to get the raw HTML code.
6. Click "Print / PDF" to print the document or save it as a PDF via the browser's print dialog.
