# Regex Tester & Visualizer Pro

A tool for creating, testing, and understanding regular expressions.

## Features

- **Real-time Highlighting**: See matches highlighted in your test string as you type.
- **Group Support**: Inspect captured groups for each match.
- **Error Reporting**: Instant feedback on invalid regex patterns.
- **Cheatsheet**: Quick reference for common regex tokens and flags.
- **Dark Mode**: Comfortable interface for long coding sessions.

## Usage

1. Open `index.html` in your browser.
2. Enter your Regular Expression in the top input field.
3. Add flags (e.g., `g`, `i`, `m`) in the small input next to it.
4. Type or paste text into the "Test String" area.
5. View the highlighted matches in the text area and detailed match information in the bottom panel.
