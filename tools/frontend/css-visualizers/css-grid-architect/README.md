# CSS Grid Layout Architect

A sophisticated visual tool for designing and understanding CSS Grid.

## Features

- **Container Controls**: Configure `grid-template-columns`, `grid-template-rows`, `gap`, `justify-items`, `align-items`, and more.
- **Item Controls**: Place items precisely using `grid-column` and `grid-row` start/end lines.
- **Live Preview**: See your grid layout come to life instantly.
- **CSS Output**: Get the exact CSS needed to recreate your grid.
- **Interactive UI**: Add/remove items and select them to tweak their positioning.

## Usage

1. Open `index.html` in your browser.
2. Define your grid tracks (columns and rows) using the sidebar inputs.
3. Adjust gap and alignment properties.
4. Click on an item to select it, then use the item controls to span it across multiple columns or rows.
5. Copy the generated CSS.
