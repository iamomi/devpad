# CSS Flexbox Playground X

An advanced visual tool for mastering CSS Flexbox.

## Features

- **Container Controls**: Full control over `display`, `flex-direction`, `flex-wrap`, `justify-content`, `align-items`, `align-content`, and `gap`.
- **Item Controls**: Select individual items to adjust `order`, `flex-grow`, `flex-shrink`, `flex-basis`, `align-self`, and dimensions.
- **Live Preview**: Real-time visualization of the flex container and items.
- **CSS Output**: Instantly generated CSS for both the container and the selected item.
- **Interactive UI**: Add/remove items dynamically, click to select items for editing.

## Usage

1. Open `index.html` in your browser.
2. Use the sidebar controls to adjust the Flexbox container properties.
3. Click on any numbered item in the preview area to select it.
4. Once selected, a new set of controls will appear in the sidebar to adjust that specific item's flex properties.
5. Copy the generated CSS from the bottom panel.
