# DevToolBox

A collection of web-based test pads, visualizers, and utilities for developers.

## Tools

### Frontend

#### CSS Visualizers
- **[CSS Flexbox Playground X](tools/frontend/css-visualizers/flexbox-playground/index.html)**: Master Flexbox layouts with interactive controls.
- **[CSS Grid Layout Architect](tools/frontend/css-visualizers/css-grid-architect/index.html)**: Design complex grid layouts visually.

#### DOM Manipulation
- **[Advanced Event Inspector Pro](tools/frontend/dom-manipulation/event-inspector/index.html)**: Visualize event propagation (bubbling/capturing) in real-time.
- **[Hyper Window.open() Test Pad X](tools/frontend/window-management/window-open-tester/index.html)**: Test `window.open()` features and behaviors.

#### Network Inspectors
- **[CORS & HTTP Headers Simulator](tools/frontend/network-inspectors/cors-tester/index.html)**: Simulate and test CORS requests and headers.

#### Storage Explorers
- **[LocalStorage & SessionStorage Explorer X](tools/frontend/storage-explorers/localstorage-manager/index.html)**: Manage web storage with CRUD and JSON editing.

#### Text Processing
- **[Regex Tester & Visualizer Pro](tools/frontend/text-processing/regex-tester/index.html)**: Test regular expressions with real-time highlighting.
- **[Markdown Previewer with Export](tools/frontend/text-processing/markdown-previewer/index.html)**: Live Markdown editor with HTML export and print support.
- **[Base64 Encoder/Decoder](tools/frontend/text-processing/base64-tool/index.html)**: Convert text and files to/from Base64.
- **[JSON Formatter & Validator](tools/frontend/text-processing/json-formatter/index.html)**: Format, validate, and minify JSON.

#### Security
- **[JWT Debugger & Decoder](tools/frontend/security/jwt-debugger/index.html)**: Decode and verify JSON Web Tokens locally.
